# ButiLab - Impressões 3D

Website institucional da ButiLab, especializada em impressões 3D.

## 📋 Sobre o Projeto

Este é o site da ButiLab, uma empresa especializada em serviços de impressão 3D. O site apresenta os serviços oferecidos, projetos realizados e informações de contato.

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura do site
- **CSS3** - Estilização e layout responsivo
- **JavaScript** - Interatividade do menu mobile
- **Video Background** - Vídeo de fundo na seção inicial

## 📱 Funcionalidades

- Design responsivo (mobile-first)
- Menu hambúrguer para dispositivos móveis
- Navegação suave entre seções
- Vídeo de fundo com efeito de blur
- Layout moderno e clean

## 🛠️ Como Executar

1. Clone este repositório
2. Abra o arquivo `public_html/index.html` no seu navegador
3. Ou use um servidor local (recomendado para melhor experiência)

## 📁 Estrutura do Projeto

```
Butilab/
├── public_html/
│   ├── index.html          # Página principal
│   ├── css/

│   │   └── menubar.js      # Menu mobile
│   ├── img/                # Imagens

```
